Installing MySQL

Install MySql depending on your distro:

Redhat/CentOS 
sudo yum install mysql-server
sudo /sbin/service mysqld start

Debian/Ubuntu 
sudo apt-get update
sudo apt-get install mysql-server
sudo service mysql start

NOTE: You may notice in the sample project that several rest plugins and web/mvc plugins have been left out from the dependencies. This is deliberate; due to out API abstraction layer, they can cause confusion and unnecessary slowness in the application.

Database Conf

Login to mysql as 'root' and create database

create database scheduler;

While in mysql shell, create user and grant privileges using login/password you just created:

GRANT ALL PRIVILEGES ON scheduler.* to \'login\'@\'localhost\' IDENTIFIED BY \'password\';
GRANT ALL PRIVILEGES ON scheduler.* to \'login\'@\'%\' IDENTIFIED BY \'password\';
flush privileges;

Installing Grails

  • Download Groovy 2.4.7 or greater from the Groovy Website and install in '/usr/share'
  • Download Grail 3.1.9 or greater from the Grails website and install in '/usr/share'
  • Add the following to your .bashrc file in your home directory (or the /etc/bash.bashrc if you are on Ubuntu/Debian):
    • export GRAILS_HOME=/usr/share/grails-3.1.9
export GROOVY_HOME=/usr/share/groovy-2.4.7
