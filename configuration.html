Installing MySQL

Install MySql depending on your distro:

Redhat/CentOS 
sudo yum install mysql-server
sudo /sbin/service mysqld start

Debian/Ubuntu 
sudo apt-get update
sudo apt-get install mysql-server
sudo service mysql start

NOTE: You may notice in the sample project that several rest plugins and web/mvc plugins have been left out from the dependencies. This is deliberate; due to out API abstraction layer, they can cause confusion and unnecessary slowness in the application.

Database Conf

  1. Login to mysql as 'root' and create database
    2. create database scheduler;
  2. While in mysql shell, create user and grant privileges using login/password you just created:
    3. GRANT ALL PRIVILEGES ON scheduler.* to \@'localhost' IDENTIFIED BY '\';
GRANT ALL PRIVILEGES ON scheduler.* to \@'%' IDENTIFIED BY '\';
flush privileges;

Installing Grails

The BeAPI™ API Framework has the following project requirements:

  • Grails 3.1.9 (or higher)
  • spring-security-rest:2.0.0.M2 (or higher)
  • spring-security-rest-gorm:2.0.0.M2 (or higher)

NOTE: You may notice in the sample project that several rest plugins and web/mvc plugins have been left out from the dependencies. This is deliberate; due to out API abstraction layer, they can cause confusion and unnecessary slowness in the application.
