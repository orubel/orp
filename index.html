What is the ORP Framework?

The ORP Framework is a Toolkit for doing Network measurements and testing extending the PerfSonar Framework.

It was built by Owen Rubel for the University of Washington High Performance Computing Dept

What Are It's Benefits?

The ORP Framework is a system of tooling built in coordination with the PerfSonar system that provides the following advantages:

  • Simple Interface
  • Dynamic Driven Charting
  • Determine Network Reliability

Simplified Network Measurements

Network measurements are automated and cached so that they can be reported on using simplified and easy to understand charting. The types of measurements provided are:

  • Network Heat Map
  • Packet Count Sent
  • Packet Count Lost
  • One-Way Delay
  • Packet Loss Rate
  • Packet Trace
  • Packet Retransmits
  • Throughput
